# JSPM - React build process

## features

- JSPM, SystemJS
- React
- SystemJS hot reload
 - script & css file
- PostCSS
- build process for publishing
- lint
 - eslint
 - stylelint

## install

```bash
npm install
jspm install
```

## run local server

```bash
gulp
```

## build

```bash
gulp build
```

## test

```bash
npm test
```
