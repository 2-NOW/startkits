# JSPM + Gulp Boilerplate

A simple Gulp automation process based on JSPM. 

## feature
JSPM, Gulp, React, ES6, Jade, Sass 

## commands

### Install 

```bash
npm install
jspm install
```


### Run

```bash
gulp
```

then connect to [localhost:7677](http://localhost:7677) and write a code.


### Build
```bash 
gulp build
```
