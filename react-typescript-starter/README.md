# react-typescript-starter

## fetures

- created with [create-react-app](https://github.com/facebookincubator/create-react-app)
- Typescript, TSX
- Redux
- HMR

## npm commands

### start development

```bash
npm run dev
```

### deploy after build

with `.env.development`

```bash
npm run deploy:test
```

with `.env.production`

```bash
npm run deploy:prod
```
